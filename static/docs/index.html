POST

/deliveries

Adds a new Delivery object

Headers

Authentication

Basic YWRtaW46cHJvdmVkb3IyMDE5

Content-Type

application/json

Body

orderId

The order ID.

clientId

The client ID.

receiver

The receiver name.

receiverCpf

The receiver CPF.

receiverIsClient

The value that indicates if receiver is the client.

location

The location for delivery.

Responses

200

The delivery object that had just been added.

400

The request missed some required fields in body.

401

The request missed or doesn't match the authentication in header.

GET

/deliveries

Lists all delivery objects added

Headers

Authentication

Basic YWRtaW46cHJvdmVkb3IyMDE5

Content-Type

application/json

Responses

200

The list of all delivery objects.

204

There are not any delivery objects added.

401

The request missed or doesn't match the authentication in header.

GET

/deliveries/:deliveryId

Fetches the matching delivery object

Headers

Authentication

Basic YWRtaW46cHJvdmVkb3IyMDE5

Content-Type

application/json

Parameters

deliveryId

The delivery id to search for.

Responses

200

The delivery object that matches the passed 'ID'.

401

The request missed or doesn't match the authentication in header.

404

The requested delivery object was not found, 'ID' may be not added.

PATCH

/deliveries/:deliveryId

Updates the matching delivery object

Headers

Authentication

Basic YWRtaW46cHJvdmVkb3IyMDE5

Content-Type

application/json

Parameters

deliveryId

The delivery ID to search for updating.

Body

orderId

The new order ID to be updated.

clientId

The new client ID to be updated.

receiver

The new receiver name to be updated.

receiverCpf

The new receiver CPF to be updated.

receiverIsClient

The new value that indicates if receiver is client.

location

The new location to be updated.

Responses

200

The delivery object have been updated sucessfully.

401

The request missed or doesn't match the authentication in header.

404

The requested delivery object was not found, 'ID' may be not added.

deliveryId

404
